Projects

Fifty Challenging Problems in Probability

Backgammon clock

backgammon-clock is simple Fischer style timer for backgammon intended to be used on a mobile device. There is a mobile friendly version of the clock here and the code is hosted here.

Epidemic simulator

epi-sim: a library for simulating epidemics, with a focus on phylodynamics and observation models.

Vega Cookbook

The vegacookbook is my attempt to port the examples from the BBC's rcookbook into vegalite. This is a work in progress, and I do not intend to get this perfect, just close enough that it provides some useful examples of making publication quality figures. If you think you can improve this, please lodge a PR.

Nicemacs

My configuration for spacemacs, nicemacs is available as a literate document. Some of it is pretty specific to spacemacs and the machines I work on, but the snippets might be of interest to emacs users.
