Projects

One Thousand Exercises in Probability

In 2021 I used the book One Thousand Exercises in Probability as a way to learn more Maxima and probability. I only considered the problems at the end of each chapter rather then the execrises throughout the chapters.

Fifty Challenging Problems in Probability

In 2021 I used the book Fifty Challenging Problems in Probability as a set of exercises to learn how to use Maxima.

White Chocolate Macadamia

A Haskell Stack template for a cookie cutter project. 

stack new my-project https://raw.githubusercontent.com/aezarebski/white-chocolate-macadamia/main/macadamia.hsfiles

Epidemic simulator

epi-sim: a library for simulating epidemics, with a focus on phylodynamics and observation models.

Vega Cookbook

The vegacookbook is my attempt to port the examples from the BBC's rcookbook into vegalite. This is a work in progress, and I do not intend to get this perfect, just close enough that it provides some useful examples of making publication quality figures. If you think you can improve this, please lodge a PR.

Nicemacs

My configuration for of spacemacs is nicemacs. You can get a copy of it from github. Some of it is pretty specific to spacemacs and the machines I work on, but the snippets might be of interest to emacs users.
