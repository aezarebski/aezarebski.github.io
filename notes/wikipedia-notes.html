Creating an article

Search for the article and if there is not a good match there will be a red link which leads to the article creation page.

Naming References

To reference the same article multiple times, on the first use wrap it in a ref tag with a name attribute, on subsequent usages you can reference it by that name in another ref tag. For example: 

Things happened<ref name="smith2000things">CITATION</ref>. Some of the things were good<ref name="smith2000things" />.

where the CITATION is the full citation curly brace business.

Citation Gadget

There is a Citation expander gadget which greatly simplifies the process of adding citations to articles. It is enabled though Preferences then Gadgets. Once it has been enabled, to add a citation of a journal article, you just need to add <ref>{{cite journal |doi=10.1000/ABC123}}</ref> and then click the Citations button. There are other stubs that can be used:

  • {{cite journal |doi=10.1000/ABC123}}
  • {{cite book |isbn=978-1-23-456789-0}}
  • {{cite web |url=https://example.com}} (questionable)

Note: bear in mind when using the citation gadget that

  • it takes a while to do its magic, but it will do magic so be patient,
  • and you still need to wrap the citation in a <ref></ref> for this to format the citation correctly.
