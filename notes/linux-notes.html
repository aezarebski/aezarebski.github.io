Mounting

The following steps seem to work…

  1. Find the drive with fdisk -l (for example /dev/sdc1)
  2. Mount the drive with mount (mount /dev/sdc1 /media/usb-drive)
  3. Use the drive
  4. Dismount with umount (umount /media/usb-drive)

Depending on how the drive is formatted you may need to install packages to work with exFAT.

i3 configuration

Here are some nifty shortcuts you might want to add to your ~/.config/i3/config file 

# start libre office
bindsym $mod+o exec libreoffice

# start a browser
bindsym $mod+b exec google-chrome

# start a mail client
bindsym $mod+m exec thunderbird

# start a document viewer
bindsym $mod+v exec evince

# start a file manager GUI
bindsym $mod+g exec nautilus --no-desktop

# lock the screen
bindsym $mod+Shift+z exec i3lock

Mouse settings

Method 1

The following commands can be used to modify the sensitivity of a mouse from the command line. 

xinput list                     # to find the ID of the mouse
xinput list-props <mouseID>     # to get the current properties of the mouse
xinput set-props <mouseID> <propID> <newValue> # update the property value

Method 2

Switch the left and right buttons on the mouse by running the following command: 

xmodmap -e "pointer = 3 2 1"

VPN and working remotely

screen

  • You need to have screen installed.
  • Start a new screen session with screen.
  • Detach from a screen with C-a d.
  • Re-attach to a screen with screen -r from the command line.
  • screen -ls will list existing screens.

The Oxford VPN

To run the VPN GUI in Ubuntu 

/opt/cisco/anyconnect/bin/vpnui

Then you need to connect to vpn.ox.ac.uk with the zool[0-9]{4} username.
