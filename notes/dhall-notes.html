Examples

REPL

dhall repl

Command line usage

dhall-to-json <<< '{ foo = [1, 2, 3], bar = True }'

Type annotations and schema

Inline type annotations 

dhall-to-json <<< '{ foo = 1, bar = True } : { foo : Natural, bar : Bool }'

Schema in a file 

echo '{ foo : Natural, bar : Bool }' > schema.dhall
dhall-to-json <<< '{ foo = 1, bar = True } : ./schema.dhall'

Prelude

Import the prelude from the URL (which can take a while…) 

⊢ :let Prelude = https://prelude.dhall-lang.org/v15.0.0/package.dhall

see what modules there are 

⊢ Prelude.<TAB>
Prelude.Bool      Prelude.JSON      Prelude.Monoid    Prelude.XML
Prelude.Double    Prelude.List      Prelude.Natural
Prelude.Function  Prelude.Location  Prelude.Optional
Prelude.Integer   Prelude.Map       Prelude.Text

Installation

  1. Get a recent version,
  2. Unpack it: tar --extract --bzip2 --file dhall-*-x86_64-linux.tar.bz2,
  3. Put it on your path sudo cp -i ./bin/dhall /usr/bin/.
