Bivariate Poisson Distribution

  • The actual idea appears to have come from Panditi et al (1988).
/* Load the package needed to simplify sums */
load(simplify_sum);

/* These functions must be bounded and are needed in the joint distribution. */
g_1(u) := exp(-u);
g_2(v) := exp(-v);

/* The marginal distributions are both Poisson. */
p_1(u) := (λ_1 ^ u) * exp(-λ_1) / factorial(u);
p_2(v) := (λ_2 ^ v) * exp(-λ_2) / factorial(v);

/* We need the expected values of the weighting g functions. */
g_1_bar : simplify_sum(sum(g_1(u) * p_1(u), u, 0, inf));
g_2_bar : simplify_sum(sum(g_2(v) * p_2(v), v, 0, inf));

/* The joint distribution is then given by the following expression */
p(u,v) := p_1(u) * p_2(v) * (1 + α * (g_1(u) - g_1_bar) * (g_2(v) - g_2_bar));

By requiring that the joint distribution is always non-negative you can show that

\[ \alpha \geq - \frac{1}{(1-A)(1-B)} \]

where \(A = e^{-\lambda_{1}c}\) and \(B = e^{-\lambda_{2}c}\) where \(c = 1 - e^{-1}\). So,

\[ \mid \alpha \mid \leq \frac{1}{(1-A)(1-B)}. \]
