External CSS

To apply CSS from a file add a link in the head to that file. 

<link rel="stylesheet" href="mystyle.css">

Internal CSS

To have inline CSS include the following element in the <head>. 

 <style>
body {
  background-color: linen;
}

h1 {
  color: maroon;
  margin-left: 40px;
}
</style>
