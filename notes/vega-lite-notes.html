Local usage

  1. Save the following snippet as a HTML file
  2. Put your specification, my-vega-lite.json, in the same directory
  3. python3 -m http.server 8000 and open the HTML with a browser
<!DOCTYPE html>
<html>
    <head>
        <title>Vega-Lite</title>
        <script src="https://cdn.jsdelivr.net/npm/vega@5.13.0"></script>
        <script src="https://cdn.jsdelivr.net/npm/vega-lite@4.14.1"></script>
        <script src="https://cdn.jsdelivr.net/npm/vega-embed@6.10.0"></script>
    </head>
    <body>
        <div id="vis"></div>

        <script type="text/javascript">
         var spec = "http://0.0.0.0:8000/my-vega-lite.json";
         vegaEmbed('#vis', spec).then(function(result) {
         }).catch(console.error);
        </script>
    </body>
</html>

A more detailed example with javascript

This figure is created in two parts: the left panel showing the average miles per gallon, and the right panel showing the horsepower. We use the standard cars data set for this. 

const carsUrl = "https://cdn.jsdelivr.net/npm/vega-datasets@1.31.1/data/cars.json"

Here we use fieldT because the year is a temporal variable and adjust the title of the y-axis to avoid having underscores in the name. In its current state, this only shows a line connecting all of the data points. 

const mpg = vl
      .markLine()
      .width(300)
      .data(carsUrl)
      .encode(vl.x().fieldT('Year'),
              vl.y().average('Miles_per_Gallon')
              .title("Average miles per gallon")
             );

This is very similar, expect we use fieldQ because the horsepower is a quantitative variable, and we also use the tooltip to display additional information about each point in the figure. 

const hp = vl
      .markPoint()
      .width(500)
      .data(carsUrl)
      .encode(vl.x().fieldT("Year"),
              vl.y().fieldQ('Horsepower'),
              vl.tooltip(["Name", "Origin"]));

We combine the figures we made above horizontally with hconcat and in the case of the miles per gallon, we create an additional layer which has circles to incidate the individual points. 

const combSpec = vl.hconcat(vl.layer(mpg,
                                     mpg.markCircle()),
                            hp).toJSON();

vegaEmbed('#vis06', combSpec);

Histogram with javascript

const dataUrl07 = "https://aezarebski.github.io/misc/cars.json";

const chartSpec07 = vl
      .markBar()
      .data(dataUrl07)
      .encode(
          vl.x().fieldN("Origin").axis({"title": ""}),
          vl.y().aggregate("count").axis({"grid": false}).title("Frequency"),
          vl.column().fieldO("Cylinders"),
          vl.color().fieldN("Origin")
      )
      .config({})
      .toJSON();

vegaEmbed('#vis07', chartSpec07);

Heatmap with javascript

const weatherCsv = "https://aezarebski.github.io/misc/seattle-weather.csv"

const weatherHeatmap = vl
      .markRect()
      .data(weatherCsv)
      .encode(
          vl.x().timeUnit("date").fieldO("date").title("Day").axis({"format": "%e"}),
          vl.y().timeUnit("month").fieldO("date").title("Month"),
          vl.color().max("temp_max").fieldQ("temp_max").legend({"title": null})
      )
      .width(500)
      .height(400)
      .title("Daily Max Temperatures (C) in Seattle, WA")
      .toJSON();

vegaEmbed("#vis08", weatherHeatmap);
