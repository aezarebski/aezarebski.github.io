1 Emacs

Frames contain windows. To open a new frame there is the new-frame command.

1.1 The daemon and the client

To start an Emacs daemon run emacs --daemon. To open a new frame run emacsclient -c, or to open a specific file emacsclient -c <path/to/file>.

2 Modes

2.1 Emacs Speaks Statistics

In spacemacs, if you are running an R process and you ? a function in the REPL it opens the documentation in emacs mode rather than evil. To fix this you can C-z to get into vim. To close the documentation in emacs mode just use q, but to do this in vim use :q.

2.2 mu for Emacs

Start the mu4e application with SPC a M. Access the manual with H from the main screen. Execute marks with x, the main ones are in the following table:

Mark Keybinding
unread ?
move m
trash d
unmark u

Further, here is a snippet from the documentation explaining the flags 

• The letters in the ‘Flags’ field correspond to the following:
  D=_draft_, F=_flagged_ (i.e., ‘starred’), N=_new_, P=_passed_
  (i.e., forwarded), R=_replied_, S=_seen_, T=_trashed_,
  a=_has-attachment_, x=_encrypted_, s=_signed_, u=_unread_.  The
  tooltip for this field also contains this information.
