Visualizing Data

The page constructed in this chapter has the following index.html and app.js files. This acts as a sensible starting point for the exercises below. 

<!DOCTYPE html>
<html>
    <head>
        <title>Visualizing Data</title>
        <meta charset="utf-8">
        <script type="module" src="app.js" async></script>
    </head>
    <body>
        <div id="vis"></div>
    </body>
</html>

import vegaEmbed from 'vega-embed';

let spec = {
    "$schema": "https://vega.github.io/schema/vega-lite/v2.0.json",
    "description": "Create data array but do not display anything.",
    "data": {
        "values": [
            {"a": "A", "b": 28},
            {"a": "B", "b": 55},
            {"a": "C", "b": 43},
            {"a": "D", "b": 91},
            {"a": "E", "b": 81},
            {"a": "F", "b": 53},
            {"a": "G", "b": 19},
            {"a": "H", "b": 87},
            {"a": "I", "b": 52}
        ]
    },
    "mark": "bar",
    "encoding": {
        "x": {"field": "a", "type": "ordinal"},
        "y": {"field": "b", "type": "quantitative"}
    }
};

let options = {
    "actions": {
        "export": true,
        "source": false,
        "editor": false
    }
};

vegaEmbed("#vis", spec, options);

npm init -y
npm install parcel vega vega-lite vega-embed
npx parcel index.html

Exercises

Binned Scatterplots

To make a binned scatter plot you can look at the example code. The only thing we need to change in our code is the replace the mark and encoding with the following. 

"mark": "circle",
"encoding": {
    "x": {
        "field": "a"
    },
    "y": {
        "field": "b"
    },
    "size": {"aggregate": "count"}
}

Grouped Bar Charts

To make a grouped bar char you cat look at the example code. The important thing to note here is that there is another variable in the data which then maps to colour and xOffset in the encoding. Both the colour and xOffset are required for the figure to work.

Note that the height of the bars in the provided example are read directly from the data, so if you want to do a histogram in this manner, another step will be needed to compute the relevant summary statistics.

Limits of Declarative Programming

There does not appear to be any 3D visualisations in the example gallery. Presumably this is because the Vega-Lite grammar does not yet have an elegant (or any) way to define such a visualisation.

Working With Arrays

We need to npm install d3-array for the following example, but with that done the rest is pretty straightforward. The index.html can stay the same, but we need to modify app.js. The first step is to import the required function and create a report of the summary statistics. 

import { deviation, mean, quantile, range } from 'd3-array';

const randVars = range(1000).map((x) => Math.random());

const newPara1 = document.createElement("p");
const newContent1 =
      newPara1.appendChild(
          document.createTextNode(
              `The mean is ${mean(randVars)}.`));

const newPara2 = document.createElement("p");
const newContent2 =
      newPara2.appendChild(
          document.createTextNode(
              `The standard deviation is ${deviation(randVars)}.`));

const newPara3 = document.createElement("p");
const newContent3 =
      newPara3.appendChild(
          document.createTextNode(
              `The quantiles are ${
[quantile(randVars, 0.25),
quantile(randVars, 0.5),
quantile(randVars, 0.75)]}.`));

const newDiv = document.createElement("div");
newDiv.append(newPara1);
newDiv.append(newPara2);
newDiv.append(newPara3);
document.body.appendChild(newDiv);

Then, copying the Vega-Lite example histogram specification we can create the desired figure. Note that because we have appended the new div above the figure still sits above the reported values. 

let spec = {
    "$schema": "https://vega.github.io/schema/vega-lite/v2.0.json",
    "data": {
        "values": randVars.map((x) => {return {val: x}})
    },
    "mark": "bar",
    "encoding": {
        "x": {
            "bin": true,
            "field": "val",
            "title": "Random variable"
        },
        "y": {
            "aggregate": "count"
        },
    }};

let options = {
    "actions": {
        "export": true,
        "source": false,
        "editor": false
    }
};

vegaEmbed("#vis", spec, options);
