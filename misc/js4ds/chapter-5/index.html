Naming

The sm in the bootstrap class names col-sm-4 and col-sm-8 indicate that this class is to be applied on tablet sized small screens. The numbers 4 and 8 indicate how much of the screen should be used for that column. There are four different screen sizes considered: xs, sm, md, and lg.

Colour

Here is a simple page which uses the main.js script to create a 10-by-10 table with the word "colour" shown in 100 randomly generated colours. 

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
    </head>
    <body>
        <div id="main"></div>
        <script src="main.js"></script>
    </body>
</html>

const randomColour = () => {
    let colour = '#';
    for (let i = 0; i < 6; i++){
        const random = Math.random();
        const bit = (random * 16) | 0;
        colour += (bit).toString(16);
    };
    return colour;
};

const colourfulTable = () => {
    let table = '';
    table += '<table>';
    for (let i = 0; i < 10; i++) {
        table += '<tr>';
        for (let j = 0; j < 10; j++) {
            table += '<td>';
            table += `<p style='color: ${randomColour()};'>colour</p>`;
            table += '</td>';
        }
        table += '</tr>';
    }
    table += '</table>';
    return table;
};

const main = document.getElementById('main');
main.innerHTML = colourfulTable();
