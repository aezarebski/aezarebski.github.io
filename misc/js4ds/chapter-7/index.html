Dynamic Pages

I couldn't get the examples in this chapter to work as described in the book, but the examples on the parceljs website worked as a starting point. The set up had the following files. 

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>Dynamic Pages</title>
  </head>
  <body>
    <div id="app"></div>
    <script type="module" src="index.js"></script>
  </body>
</html>

The index.js script had the following 

import ReactDOM from "react-dom";

const app = document.getElementById("app");
ReactDOM.render(<h1>Hello world!</h1>, app);

Setting this up with node required the following commands 

npm init -y
npm install react react-dom parcel
npx parcel index.html

From this point it was reasonable simple to adapt the code provided in the chapter.

Exercises

Those Damn Curly Braces

In JSX, curly braces are used to indicate that their contents should be parsed as a javascript expression rather than a literal string intended to be included in the page. If they are removed, the expression they contain will be treated as the text of that tag.

Real Data

We can extend index.js from above to read in a variable exported from a module called programmers. 

import ReactDOM from "react-dom";

import { programmers } from "./programmers.js";

const app = document.getElementById("app");

const ListElement = (props) =>
      (<li><em>{props.firstName}</em></li>);

ReactDOM.render(
        <ul>{programmers.map((programmer) =>
                <ListElement firstName={programmer.firstName}
            lastName={programmer.lastName} /> )}</ul>,
    app
);

and the file that we import the data from is the following 

const programmers = [
    {firstName: "Donald", lastName: "Knuth"},
    {firstName: "Richard", lastName: "Stallman"}
];

export { programmers }

Ordering

Multiple Targets

If there are multiple div elements with id=app then the first one is used as the target.

Creating a Component for Names

We can adjust the end of the index.js script to use a new Name component. 

const Name = (prop) =>
      (<em>{prop.firstName + " " + prop.lastName}</em>);

ReactDOM.render(
        <ul>{
            programmers.map(
                (p) => <li>
                    <Name
                firstName={p.firstName}
                lastName={p.lastName}/>
                    </li>)
        }</ul>,
    app
);

Striping

The map method allows for another argument which is the index of the element. We can use this to determine if the element should be in italics and add another attribute to the Name component to specify when this should be used. 

const Name = (prop) => {
    const name = prop.firstName + " " + prop.lastName;
    if (prop.it % 2 === 1) {
        return (<em>{name}</em>);
    } else {
        return (<p>{name}</p>);
    }
};

ReactDOM.render(
        <ul>{
            programmers.map(
                (p,ix) => <li>
                    <Name
                firstName={p.firstName}
                lastName={p.lastName}
                it={ix} />
                    </li>)
        }</ul>,
    app
);
