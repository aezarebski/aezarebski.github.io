Functional programming

The Array module has the basic functions for working with collections:

  • Array.some
  • Array.every
  • Array.filter
  • Array.map
  • and for ordered collections Array.reduce

Exercises

Side Effects With forEach

A solution is given in the following script 

doubleInPlace = (xs) => {
    return xs.forEach((x, ix, array) => {
	array[ix] = 2 * x;
    });
};

const vals = [1, 2, 3];

console.log(`vals before change: ${vals}`);
doubleInPlace(vals);
console.log(`vals after change: ${vals}`);

which when run performs the following action 

$ node foo.js
vals before change: 1,2,3
vals after change: 2,4,6

Annotating Data

The suggestion to use reduce is actually a bit of a distraction; the map method will supply the index if there are two parameters for the given function. This means we can get the result with a simple filter followed by a map. 

const input = [
    {'date': '1977-7-16', 'sex': 'M', 'species': 'NL'},
    {'date': '1977-7-16', 'sex': 'M', 'species': 'NL'},
    {'date': '1977-7-16', 'sex': 'F', 'species': 'DM'},
    {'date': '1977-7-16', 'sex': 'M', 'species': 'DM'},
    {'date': '1977-7-16', 'sex': 'M', 'species': 'DM'},
    {'date': '1977-7-16', 'sex': 'M', 'species': 'PF'},
    {'date': '1977-7-16', 'sex': 'F', 'species': 'PE'},
    {'date': '1977-7-16', 'sex': 'M', 'species': 'DM'}
];

const output = input
      .filter(x => x.sex == 'F')
      .map((x, i) => {
	  x.seq = i+1;
	  x.year = x.date.split('-')[0];
	  delete x.date;
	  return x;
      });
console.log(output);

which when we run gives 

$ node foo.js
[
  { sex: 'F', species: 'DM', seq: 1, year: '1977' },
  { sex: 'F', species: 'PE', seq: 2, year: '1977' }
]
