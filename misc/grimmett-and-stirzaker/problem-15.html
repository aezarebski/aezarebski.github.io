\(N\) dice are thrown where \(\mathbb{P}(N=n) = 2 ^{-n}\) for \(n\geq1\). The sum of the dice is \(S\). Find the probability that:

a.) \(N=2\) given \(S=4\); b.) \(S=4\) given \(N\) is even; c.) \(N=2\) given \(S=4\) and the first die showed 1; d.) the largest number shown by any dice is \(r\).

First lets define the distribution for a single die and dice. 

load(distrib);
load(simplify_sum);

pmf_dice(x) := pdf_discrete_uniform(x, 6);

pmf_sum_dice(x, n) :=
if is(equal(n, 1))
then pmf_dice(x)
else sum(pmf_dice(i) * pmf_sum_dice(x - i, n - 1), i, 1, 6);

Then we can use this to define a couple of useful distributions for our particular problem. 

pmf_N(i) := if is(i >= 1) then 2^(-i) else 0;

pmf_joint(n, s) := pmf_N(n) * pmf_sum_dice(s, n);

pmf_S(s) := sum(pmf_joint(n, s), n, 1, s);

pmf_N_given_S(n, s) := pmf_joint(n, s) / pmf_S(s);

Part A

sol_a : (1/12 * 1/4) / (1/6 * 1/2 + 1/12 * 1/4 + 3/216 * 1/8 + 1/(6^4) * 1/16);
is(equal(pmf_N_given_S(2, 4), sol_a));

Part B

prob_N_even : simplify_sum(sum(pmf_N(2 * n), n , 1, inf));
prob_S_4_given_N_even : sum(pmf_joint(2 * n, 4), n, 1, 2) / prob_N_even;
sol_b : (4 ^ 2 * 3 ^ 3 + 1)  / (4 ^ 4 * 3 ^ 3);
is(equal(prob_S_4_given_N_even, sol_b));

Part C

a : pmf_dice(3) * pmf_dice(1) * pmf_N(2);
b : sum(pmf_dice(1) * pmf_sum_dice(3, n) * pmf_N(n+1), n, 1, 3);
sol_c : (1/6 * 1/6 * 1/4) / (1/6 * 1/6 * 1/4 + 1/6 * 2/36 * 1/8 + 1/(6^4) * 1/16);
is(equal(a / b, sol_c));

Part D

  • The probability that \(n\) dice are \(\leq r\) is \((r/6)^n\).
  • Summing over \(n\) (using a geometric series,) the probability that the maximum is \(\leq r\) is \(r / (12 - r)\).
  • The probability that the maximum is equal to \(r\) is the probabiltiy it is \(\leq r\) minus the probability is is \(\leq r-1\).
