Let \(A_1,A_2,\dots\) be a sequence of events. Define

\[ B_n = \bigcup_{m=n}^{\infty} A_m, \quad C_n = \bigcap_{m=n}^{\infty} A_m. \]

and

\[ B = \lim B_n = \bigcap_n B_n, \quad C = \lim C_n = \bigcup_n C_n \]

Show that

a.) \(B = \{ \omega\in\Omega : \omega \in A_n \text{ for infinitely many values of } n \}\)

b.) \(C = \{ \omega\in\Omega : \omega \in A_n \text{ for all but finitely many values of } n \}\)

if \(B=C\) then we say \(A_n\to A\). If \(A_n\to A\) show that

c.) \(A\) is an event

d.) \(\mathbb{P}(A_n) \to \mathbb{P}(A)\)

Part A

LHS is a subset of RHS

If \(\omega\in B\) then it is in all the \(B_n\), hence for any number \(N\), \(\omega\in A_m\) for some \(m > N\). Therefore \(\omega\) must be in infinitely many of the \(A_n\) because otherwise there would be an \(N\) that violates this.

RHS is a subset of LHS

If \(\omega\in A_n\) for infinitely many \(n\) then \(\omega\in B_n\) for all \(n\), hence it is in \(B\).

Part B

LHS is a subset of RHS

If \(\omega\in C\) then \(\omega\in C_m\) for some \(m\). This means that \(\omega\in A_k\) for all \(k \geq m\). Hence \(\omega\) can only be missing from a finite number of \(A_n\).

RHS is a subset of LHS

If \(\omega\in A_n\) for all but a finite number of \(n\), then there is a smallest \(k\) such that \(\omega\in A_n\) for all \(n\geq k\). Then \(\omega\in C_k\) and so \(\omega\in C\).

Part C

By definition, countable unions are events. From problem 1.8.3 we know countable intersections are events. Hence \(B\) is an event and so is \(A\).

Part D

Note that \(\mathbb{P}(C_n) \leq \mathbb{P}(A_n) \leq \mathbb{P}(B_n)\) and apply the Squeeze theorem.
